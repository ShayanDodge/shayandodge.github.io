Lightning Geolocation & Peak Current Estimation. [[Github](https://github.com/ShayanDodge/FELINES-Lightning-Forecast/tree/main/Phase%201%20%E2%80%94%20Lightning%20Geolocation%20and%20Peak%20Current%20Estimation)]

We are pleased to announce the release of the GitHub repository for Version v1.0.0 of the FELINES project.

This version focuss on:

- Lightning strike geolocation (x, y estimation)
- Channel-base peak current regression
- Deep learning–based modeling of lightning-induced voltages on overhead transmission lines
- Induced Voltage Waveforms → {Strike Location (x, y), Peak Current}

This implementation reproduces and extends the methodology presented in:

  𝘌𝘭𝘦𝘤𝘵𝘳𝘪𝘤 𝘗𝘰𝘸𝘦𝘳 𝘚𝘺𝘴𝘵𝘦𝘮𝘴 𝘙𝘦𝘴𝘦𝘢𝘳𝘤𝘩, 2025. [[Link](https://scholar.google.com/citations?view_op=view_citation&hl=en&user=1usufq0AAAAJ&citation_for_view=1usufq0AAAAJ:9yKSN-GCB0IC)]
